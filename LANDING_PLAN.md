## 1. 프로덕트 가치 분석

### 1.1 핵심 가치 제안 (Core Value Proposition)

**"콘텐츠 발행 전, 실제 타겟 고객의 반응을 미리 확인하세요"**

| 문제 | Percepta 솔루션 |
|------|-----------------|
| 콘텐츠 성과 예측 불가 | AI 페르소나가 타겟별 반응 시뮬레이션 |
| 피드백 수집에 시간/비용 과다 | 실시간 피드백, 건당 ₩10 |
| 주관적 직감에 의존 | 감성 분석, 구매 의향, 점수로 정량화 |
| FGI/설문 진행 부담 | 클릭 몇 번으로 5명 이상 피드백 |

### 1.2 경쟁 우위

1. **가격**: 피드백 1건 ₩10 (경쟁사 대비 1/10~1/20 수준)
2. **속도**: 실시간 피드백 (FGI 대비 수일 단축)
3. **다양성**: 무제한 페르소나 조합 가능
4. **정량화**: 감성/구매의향/점수를 수치로 제공

### 1.3 해결하는 핵심 Pain Point

```
"내 콘텐츠가 타겟 고객에게 어떻게 느껴질지 모르겠다"
```

- 마케터: "이 광고 카피가 2030 여성에게 먹힐까?"
- 창업자: "이 랜딩페이지로 전환이 될까?"
- 콘텐츠 크리에이터: "이 글이 독자에게 공감될까?"
- 기획자: "이 기획안을 상사가 좋아할까?"

## 2. 타겟 유저 분석 (결제 전환 가능성 높은 세그먼트)

### 2.1 결제 가능성 높은 유저 우선순위

| 순위 | 세그먼트 | 결제 확률 | 이유 |
|------|----------|-----------|------|
| 1위 | **1인 마케터/퍼포먼스 마케터** | 매우 높음 | 성과=수입, 실패 비용 체감 극대화 |
| 2위 | **스타트업 마케팅 담당** | 높음 | 예산 제한 + 빠른 실험 필요 |
| 3위 | **프리랜서 카피라이터/디자이너** | 높음 | 클라이언트 설득 도구로 활용 |
| 4위 | 콘텐츠 크리에이터 (유튜버/블로거) | 중간 | 피드백 욕구 있지만 지불의향 불확실 |
| 5위 | 대기업 마케팅팀 | 중간-낮음 | 의사결정 느림, 기존 리서치 존재 |

### 2.2 Primary Target: 1인 마케터/퍼포먼스 마케터

**프로필**
- 나이: 25-35세
- 역할: 광고 소재 기획, SNS 콘텐츠 운영
- 성과 지표: CTR, CVR, ROAS
- 월 콘텐츠 생산량: 20-50개

**Pain Point 깊이 분석**

| Pain Point | 강도 | 빈도 | 지불 의향 |
|------------|------|------|-----------|
| 광고 소재 성과 예측 불가 | 극심 | 매일 | 높음 |
| A/B 테스트 전 사전 필터링 필요 | 강함 | 주 2-3회 | 중간 |
| 클라이언트 보고용 근거 필요 | 강함 | 주 1회 | 중간 |
| 직감 의존 → 자신감 부족 | 중간 | 항상 | 낮음 |

**결제 트리거**
1. "이 소재로 광고 돌려도 될까?" 순간
2. 실패한 광고 후 "미리 알았으면..." 후회
3. 클라이언트에게 "왜 이 소재인가" 설명 필요 시

**기대 효과 (유저 관점)**
- 광고비 낭비 감소 (소재당 ₩5만~50만)
- 의사결정 자신감 향상
- 클라이언트 신뢰도 상승

### 2.3 Secondary Target: 스타트업 마케팅 담당

**프로필**
- 나이: 27-38세
- 역할: 올라운더 마케터 (1인 또는 소수)
- 예산: 제한적 (FGI/설문 여력 없음)
- 콘텐츠: 랜딩페이지, 앱스토어 설명, 이메일

**Pain Point 깊이 분석**

| Pain Point | 강도 | 빈도 | 지불 의향 |
|------------|------|------|-----------|
| 리서치 예산 부족 | 극심 | 항상 | 높음 |
| 피봇 전 검증 필요 | 강함 | 월 1-2회 | 중간 |
| 타겟 고객 목소리 부재 | 강함 | 항상 | 중간 |

**결제 트리거**
1. 신규 캠페인/제품 런칭 직전
2. 전환율이 안 나올 때
3. 투자자/대표에게 "고객 반응" 보여줘야 할 때

### 2.4 결제 심리 분석

**결제까지 이어지는 심리 흐름**

```
[인지] 피드백 필요성 인식
    ↓
[탐색] 기존 방법(동료, 커뮤니티, FGI) 비교
    ↓
[발견] Percepta 발견 → "이게 되네?"
    ↓
[체험] 무료 크레딧으로 실제 사용
    ↓
[확신] "내 타겟에 맞는 피드백이 나온다"
    ↓
[결제] "이 가격이면 계속 쓰지"
```

**결제 전환 핵심 요인**

1. **즉각적 가치 체감** (무료 체험에서 "오" 느낌)
2. **신뢰성 확보** (피드백이 그럴듯해 보임)
3. **저렴한 가격** (실패해도 부담 없음)
4. **반복 사용 욕구** (한 번 쓰고 끝이 아님)

## 3. 극초기 랜딩페이지 전략

### 3.1 MVP 랜딩페이지 목표

```
"첫 방문에서 이메일/구매 의향 확보"
```

**핵심 지표**
- 목표: 5명의 결제 의향자 확보
- 전환 목표: 방문자 → 웨이트리스트/체험 신청 10%

### 3.2 페이지 구조 (위→아래)

```
┌─────────────────────────────────────────┐
│  [Hero] 한 줄 후킹 + CTA              │
├─────────────────────────────────────────┤
│  [Problem] 공감 유발 (3가지 페인)      │
├─────────────────────────────────────────┤
│  [Demo] 실제 피드백 예시 (Before/After) │
├─────────────────────────────────────────┤
│  [How] 3단계 사용 방법                 │
├─────────────────────────────────────────┤
│  [Pricing] 심플한 가격표               │
├─────────────────────────────────────────┤
│  [CTA] 무료 체험 or 웨이트리스트       │
└─────────────────────────────────────────┘
```

### 3.3 각 섹션 상세 설계

#### Hero Section

**헤드라인 후보** (A/B 테스트용)

| 옵션 | 헤드라인 | 타겟 |
|------|----------|------|
| A | "광고 소재, 돌리기 전에 검증하세요" | 퍼포먼스 마케터 |
| B | "당신의 콘텐츠, 타겟 고객은 어떻게 느낄까요?" | 일반 마케터 |
| C | "FGI 없이 5분 만에 고객 피드백 받기" | 스타트업 |

**서브헤드라인**
> AI 페르소나가 당신의 타겟 고객 시점에서 솔직한 피드백을 드립니다.
> 피드백 1건 ₩10, 지금 무료로 체험하세요.

**CTA 버튼**
- Primary: "무료로 피드백 받기" (30크레딧 = 5명 피드백)
- Secondary: "데모 보기" (스크롤 유도)

#### Problem Section

**"이런 고민, 있으신가요?"**

| 아이콘 | 문제 | 설명 |
|--------|------|------|
| 타겟 | 타겟 반응을 모르겠다 | "이 카피가 20대 여성에게 먹힐까?" |
| 비용 | 실패하면 손해가 크다 | "광고비 50만원 날릴 수도..." |
| 시간 | FGI/설문은 너무 무겁다 | "시간도, 돈도, 인력도 부족" |

#### Demo Section (핵심!)

**실제 피드백 예시 보여주기**

```
┌──────────────────────────────────────────────────┐
│ [입력] 당신의 콘텐츠                              │
│ ┌──────────────────────────────────────────────┐ │
│ │ "새해에는 건강한 습관으로 시작하세요.          │ │
│ │  지금 가입하면 첫 달 50% 할인!"               │ │
│ └──────────────────────────────────────────────┘ │
│                      ↓                           │
│ [출력] AI 페르소나 피드백                        │
│ ┌──────────────────────────────────────────────┐ │
│ │ 민지 (28세, 마케터, ENFP)                     │ │
│ │ ────────────────────────────────────────────  │ │
│ │ "새해 할인은 너무 흔해서 눈에 안 들어와요.    │ │
│ │  구체적인 혜택이나 차별점이 있으면 좋겠어요"  │ │
│ │                                               │ │
│ │ 감성: 중립  |  구매의향: 낮음  |  점수: 2.5   │ │
│ └──────────────────────────────────────────────┘ │
│ ┌──────────────────────────────────────────────┐ │
│ │ 성준 (35세, 개발자, ISTJ)                     │ │
│ │ ────────────────────────────────────────────  │ │
│ │ "50% 할인이면 괜찮은데, '건강한 습관'이       │ │
│ │  뭔지 구체적으로 알고 싶어요."                │ │
│ │                                               │ │
│ │ 감성: 긍정  |  구매의향: 중간  |  점수: 3.2   │ │
│ └──────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────┘
```

**이 섹션의 역할**
- "아, 이런 식으로 피드백이 오는구나" 직관적 이해
- "내 콘텐츠도 해보고 싶다" 욕구 자극
- 품질에 대한 신뢰 형성

#### How It Works

```
① 콘텐츠 입력          ② 페르소나 선택       ③ 피드백 확인
텍스트/URL/파일    →   5명 이상 선택     →   실시간 결과
```

**강조 포인트**
- "5분이면 끝"
- "회원가입만 하면 30크레딧 무료"

#### Pricing Section

**심플하게, 부담 없게**

```
┌─────────────────────────────────────────┐
│           피드백 1건 = ₩10              │
│                                         │
│   [무료 체험]     [기본]     [대량]     │
│    30크레딧       200크레딧   500크레딧  │
│     ₩0           ₩2,000     ₩4,500     │
│                                         │
│   * 페르소나 5명 피드백 + 요약 = 6크레딧 │
│   * 즉, ₩60으로 5명 피드백 가능!        │
└─────────────────────────────────────────┘
```

#### Final CTA Section

**긴급성/희소성 추가**

> 지금 가입하면 30크레딧 무료!
> (선착순 100명 한정)

**버튼**
- "무료로 시작하기"
- (서비스 미오픈 시) "웨이트리스트 등록" + 이메일 입력

## 4. 첫 5명 고객 획득 전략

### 4.1 채널별 접근 전략

| 채널 | 타겟 | 액션 | 예상 전환 |
|------|------|------|-----------|
| **마케터 커뮤니티** | 퍼포먼스 마케터 | 사례 공유 글 + 무료 체험 제공 | 높음 |
| **스타트업 슬랙** | 스타트업 마케터 | 문제-해결 스토리텔링 | 중상 |
| **링크드인** | B2B 타겟 | 개인 스토리 + 도구 소개 | 중상 |
| **X (트위터)** | 테크 얼리어답터 | 데모 영상/GIF | 중간 |

### 4.2 초기 고객 확보 스크립트

**커뮤니티 글 예시**

```
[공유] 광고 소재 검증할 때 쓰는 도구 만들었습니다

안녕하세요, 저도 마케터로 일하면서
"이 소재가 진짜 먹힐까?" 고민이 많았는데요.

AI로 타겟 페르소나를 만들어서
콘텐츠 반응을 미리 시뮬레이션하는 도구를 만들었습니다.

5명 피드백 받는 데 ₩60, 1분도 안 걸려요.

베타 테스터분들께 30크레딧(5회 피드백) 무료로 드립니다.
관심 있으시면 DM 주세요!
```

### 4.3 성공 지표

| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| 웨이트리스트 가입 | 50명 | 이메일 수집 |
| 체험 전환율 | 20% | 가입 → 피드백 생성 |
| 결제 전환율 | 10% | 무료 소진 → 결제 |
| **유료 고객** | **5명** | 실제 결제 완료 |

### 4.4 피드백 루프

```
1주차: 랜딩 배포 + 커뮤니티 홍보
       ↓
2주차: 초기 유저 피드백 수집
       ↓
3주차: 랜딩/UX 개선
       ↓
4주차: 5명 결제 고객 달성
```
